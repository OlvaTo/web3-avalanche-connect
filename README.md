# Avalanche Wallet Connection

Simple web project for connecting browser wallets Core Wallet to the **Avalanche Fuji Testnet**.

This project demonstrates basic wallet connection flow, network validation, and displaying wallet information using vanilla JavaScript.

---

## Features

- Connect wallet (MetaMask / Core Wallet)
- Display wallet address
- Detect active network
- Show AVAX balance
- Network validation (Fuji Testnet)
- Frontend disconnect (UI state reset)
- Wallet event handling
